Lesson 1: The beautiful Grid System

Always have a containing element/parent selector to house your display: grid. This is where you need specify the properties; grid-template-column and grid-template-rows.

Note that the code snippet below for the class grid-wrapper selector have the property grid-template-columns with the repeat() value. This simple syntax will automatically adjust the grid by filling-up its parent container.

.grid-wrapper {display:grid;grid-gap:20px;grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));grid-template-rows: 1fr;}
